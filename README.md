# Smart File Organizer

A simple command-line tool to organize your downloads folder based on file extensions and find duplicates using SHA256.

## Features

- **Organize Downloads Folder**: Automatically sorts files into folders based on their extension.
- **Duplicate Finder**: Identifies duplicate files by comparing them using SHA256 hash values.
- **Logging**: Tracks the operations performed in a log file for debugging purposes.

## Skills

- `os`
- `pathlib`
- `hashlib`
- `shutil`
